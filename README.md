# Windowed-Program

## 模拟微信抢红包（当前工程）

当前 `main.cpp` 已实现“模拟微信抢红包”窗口程序：

- 窗口标题：`模拟微信抢红包`
- 左侧展示区：
  - 红包封面图片（资源位图 `IDB_PACKET_COVER`）
  - 多行日志输出框（记录抢包和查看结果）
  - 静态文本 `Static`
- 右侧功能区：
  - 红包A分组：可输入昵称后抢红包/查看（红包封面点击会抢“当前查看”的红包）
  - 红包B分组：可输入昵称后抢红包/查看
  - 红包C分组：先输入金额与红包数塞钱，再启用“抢红包”
  - 机器人抢红包按钮：仅抢“当前查看”的红包（在A界面不抢B，在B界面不抢A）
  - 抢红包、没抢到、塞钱结果使用 MsgBox 中文提示
  - “查看”按钮切换日志为当前红包明细：用户金额列表 + 手气最佳 + 当前进度
  - 限制同一用户名在同一个红包里只可抢一次

程序中包含 `RedPacket` 类（`RedPacket.h/.cpp`）：

- `setMoney(double, int)`：塞钱（已抢后拒绝修改）
- `grab(std::string)`：二倍均值随机算法抢红包，精确到分
- `grab(std::string, int*)`：支持返回状态（成功/抢完/重复抢）
- `show()` / `summary()`：查看明细（发红包者、总份数、已抢份数、记录）
- `bestLuckRecord()`：返回手气最佳记录（用户名:金额）

## 构建运行

1. 用 Visual Studio 打开并构建 `VC2010简单Windows程序模板.sln`
2. 运行程序即可体验红包模拟流程
